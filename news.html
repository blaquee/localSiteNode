Touch event support added.
(12-06-12) Touch event support has been added. Users with touch devices can drag list items around, resize the tabs box, etc. Thanks to the Touch Punch script for jQuery-UI. The draggable items are pretty much just for fun. Mess things up and it will all straighten out on your next visit :)
New update has been applied.
(12-05-12) I've updated the website with various improvements. Shares list can now be filtered. Custom styles have been tuned a little more. Various improvements on the host side. Users who previously couldn't view content should now be able to view the site correctly. Some preparations have been made to gear up for CMS-like page management. Development continues.
Early version of new homepage is live!
(12-03-12) While this is still a fairly simple page so far, definitely in need of some shine and content, its enough to get started. I'm currently using Node.js 0.8.15 with jQuery 1.8.3 and jQuery-UI 1.9.2. The realtime capabilities available will allow for some interesting features later on. For now the goal is to find a design that works for all devices and is simple & efficient to use. Construction continues.
